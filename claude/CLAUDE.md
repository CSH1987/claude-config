# 전역 세션 기본값 (모든 프로젝트 · 모든 머신)

이 파일은 dotfiles 로 관리되며 `~/.claude/CLAUDE.md` 로 배포되어 모든 프로젝트의 모든 세션에 자동 로드됩니다.

## 작업 강도 기본값 (최우선)

- 이 사용자의 기본 모드는 **최고 강도(xhigh) + ultracode/ultraplan** 스타일입니다. 모든 작업에 적용됩니다.
- **영구 적용되는 부분**: `effortLevel: "xhigh"` 가 `~/.claude/settings.json` 에 설정돼 있어 모델 추론 강도는 매 세션 자동으로 xhigh 입니다. (실제 설정값으로 진짜 영구화됨. Opus 4.7/4.8·Fable 5 필요)
- **자동 실행되는 부분**: `claude` 명령이 셸에서 `--settings ~/.claude/ultracode.json` 으로 오버라이드돼 있어, 새 세션은 보통 **ultracode 가 이미 켜진 상태**로 시작됩니다. 이 경우 시스템 리마인더에 `Ultracode is on` 이 표시됩니다 → **추가 조치 불필요**.
- **영구화가 불가능한 부분**: ultracode 의 *동적 워크플로 오케스트레이션* 과 ultraplan 은 Claude Code 설계상 세션 전용입니다 (settings.json·환경변수·훅으로 영구화 불가).
  - 만약 이 세션이 ultracode 가 아니라면(오버라이드 미적용 환경 등), 본격적인 작업(특히 복잡/다단계/대규모)을 시작하기 전에 먼저 이렇게 제안하세요:
    > "이 세션을 ultracode(동적 워크플로 오케스트레이션)로 올릴까요? `/effort ultracode` 를 실행하면 됩니다. effortLevel=xhigh 는 이미 영구 적용 중입니다."
  - 동의하면 사용자가 직접 `/effort ultracode` 를 실행해야 합니다 (훅·Claude 는 슬래시 명령을 대신 실행할 수 없음).
- 단순/사소한 작업은 굳이 제안하지 말고 그대로 진행하세요 (불필요한 오케스트레이션 비용 회피).

## 계획 우선 (ultraplan)

## Harness 자동화

- harness 플러그인이 활성화돼 있어야 합니다. SessionStart 훅(ensure-harness)이 자동 설치/복구합니다.
- 누락이 감지되면 사용자에게 알리고, 설치 후 진행하세요.

## 사용자가 잊더라도 (능동적 보장)

- 매 세션 시작 시 위 항목들을 사용자가 먼저 말하지 않아도 능동적으로 점검·반영하세요.
- 단, 한 세션에서 이미 제안/설정/처리했다면 반복하지 마세요.
