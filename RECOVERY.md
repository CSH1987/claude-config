# 🆘 이게 다 뭐예요? — 1페이지 안내 (다 잊어버려도 OK)

> 이 한 장만 읽으면 됩니다. 어려운 말은 없습니다. 천천히 위에서 아래로.

---

## 1. 이 PC가 당신을 위해 *자동으로* 하는 일 (외울 필요 없음)

- 🧠 **기억해요.** 당신의 선호·결정·작업을 저장해, 같은 걸 두 번 설명 안 해도 됩니다.
- 🔄 **최신으로 맞춰요.** 새 터미널을 열 때마다 설정을 알아서 최신화합니다.
- ☁️ **백업해요.** 중요한 건 클라우드(GitHub)에도 복사돼, PC가 고장나도 안 사라집니다.

→ 그래서 당신은 **아무것도 기억 안 해도** 됩니다. 시스템이 대신 기억합니다.

---

## 2. 다 잃어버렸을 때 (새 PC·포맷·"이게 뭐였지?")

**터미널에 아래 한 줄**만 붙여넣으면 전부 되살아납니다. 그게 전부예요.

- 🍎 **Mac / 🐧 Linux**
  ```bash
  curl -fsSL https://raw.githubusercontent.com/<gh_handle>/claude-config/main/bootstrap.sh | bash
  ```
- 🪟 **Windows** (PowerShell)
  ```powershell
  irm https://raw.githubusercontent.com/<gh_handle>/claude-config/main/bootstrap.ps1 | iex
  ```

> `<gh_handle>` = 당신의 GitHub 사용자명. 모르면 README.md 맨 위를 보세요. 끝나면 **새 터미널**을 열고 `claude`.

---

## 3. 막히면 — Claude에게 *이렇게* 물어보세요 (복붙해도 됨)

말로 물어보면 Claude가 알아서 찾아줍니다. 명령어 외울 필요 없어요.

- 💬 "내 설정·선호가 뭐였는지 알려줘"
- 💬 "지난 결정들 보여줘" (또는 "내가 전에 뭐라고 정했지?")
- 💬 "내 기억(메모리) 백업이 어디 있는지, 안전한지 확인해줘"
- 💬 "이 시스템이 어떻게 작동하는지 1분만에 설명해줘"
- 💬 "오늘 내가 뭐부터 하면 좋을지 브리핑해줘"

---

## 4. 어디에 뭐가 있나 (한눈에)

| 무엇 | 어디 | 공개? |
|---|---|---|
| **설정·규칙·도구** (남에게 줘도 되는 것) | `claude-config` (GitHub 공개 레포) | 🌍 공개 |
| **내 기억·프로필·결정** (나만의 것) | `claude-memory` (`$CLAUDE_MEMORY_DIR`) | 🔒 비공개 |
| **성장 현황** (얼마나 쌓였나) | `metrics.md` (비공개 폴더 안) — "성장 대시보드 보여줘" | 🔒 비공개 |

> 원칙: **개인정보는 절대 공개로 안 나갑니다.** 규칙·도구만 공개됩니다.

---

## 5. 켜고 끄기 (필요할 때만)

- 끄기(잠깐 조용히): 환경변수 `CLAUDE_EVENTS_OFF=1` → 자동 기록/브리핑 멈춤.
- 다시 켜기: 그 변수를 지우고 새 터미널.
- 전체 갱신: 새 터미널에서 §2의 한 줄을 다시 실행.
